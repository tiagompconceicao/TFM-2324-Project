# TFM-2324-Project

## Overview

The TFM-2324-Project is a Proof of Concept developed as part of a Master's thesis. This project integrates a backend and a frontend, facilitating user interaction with Large Language Models (LLMs) and enabling the instantiation of containers based on the artifacts generated by these models.

## Features

- **Backend**: Developed primarily in Kotlin and Java, handling core application logic and services.
- **Frontend**: Built with React.js, providing an intuitive user interface.
- **LLM Interaction**: Users can interact with LLMs through the interface.
- **Container Instantiation**: Allows creation of containers based on model-generated artifacts.

## Structure

- **`jvm` Directory**: Contains backend services written in Kotlin and Java.
- **`node` Directory**: Frontend code developed using React.js.
- **`README.md`**: Provides an overview of the project.
